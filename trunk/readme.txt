CorvusPay payment plugin for WooCommerce that provides all popular payment methods

== Description ==

This payment plugin nicely integrates with WooCommerce and CorvusPay Payment Gateway, and provides credit card payments with
all popular credit card brands. Since this plugin uses CorvusPay as Payment Gateway, merchants that use this plugin must be Corvus Pay 
customers. For more info about CorvusPay visit http://www.corvuspay.hr/.

NOTICE!
Currently this plugin only handles preauthorization which means that completion of transaction
is handled through CorvusPay Merchant Portal.


== Installation ==

1. Install WooCommerce if it is not already installed.
1. Download and install this plugin as you would install every other WordPress plugin.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to WooCommerce > Settings > Payment Gateways > CorvusPay and populate all the values as noted below

* enable checkbox - enable this checkbox to enable the plugin in WooCommerce.
* store ID field - value that is under CorvusPay Merchant Portal identified as point of sale ID.
* CorvusPay key field - value that is generated by CorvusPay Merchant Portal and located in shop settings Key field.
* debug mode checkbox - when enabled all transactions are simulated. 
* Checkout page ID field - this value is the ID of the WordPress page that contains order checkout shortcode. This is auto-populated but can be changed.
* Success page ID field - this value is the ID of WordPress page that contains order completion shortcode. This is auto-populated but can be changed
* Cancel page ID field - this value is the ID of WordPress page the contains order cancelation shortcode. This is auto-populated but can be changed. 
* Success redirect URL field - this value is generated by the plugin and should be copied to CorvusPay Merchant Portal "success url" field. 
* Cancel redirect URL field - this value is generated by the plugin and should be copied to CorvusPay Merchant Portal "cancel url" field.
